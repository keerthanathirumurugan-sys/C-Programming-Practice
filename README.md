# Day 01 - C Programming Basics

## Objective
Learn the fundamentals of C programming and build a strong foundation for Embedded Systems and Firmware Engineering.

## Programs Completed

1. Hello World
2. Largest of Two Numbers
3. Largest of Three Numbers
4. Area of Circle
5. Average of Three Numbers
6. Even or Odd
7. Positive or Negative
8. Sum of Two Digits
9. Swapping Two Numbers

## Concepts Learned

- Structure of a C Program
- Variables and Data Types
- Input and Output (`printf`, `scanf`)
- Arithmetic Operators
- Relational Operators
- Logical Operators
- Conditional Statements (`if`, `if-else`)
- Basic Problem Solving

## Tools Used

- C
- Git
- GitHub

## Learning Outcome

- Understood the basic syntax of C.
- Wrote and executed simple C programs.
- Practiced decision-making using `if` statements.
- Improved logical thinking through programming exercises.
